# Comparable Mapping Framework — Video Analysis Agent

## Purpose
Map the target actor onto comparable performers to ground assessments in industry precedent.

## Mapping Dimensions

### Career Archetype
- **Meteoric Rise** — Sudden breakthrough, sustained peak (e.g., Jennifer Lawrence post-Winter's Bone)
- **Slow Build** — Gradual accumulation, late peak (e.g., Jessica Chastain)
- **Child Star Transition** — Navigating adult career (e.g., Zendaya, Saoirse Ronan)
- **Comeback Narrative** — Fall and resurgence (e.g., Robert Downey Jr., Brendan Fraser)
- **Consistent Artisan** — Never huge, always working (e.g., Paul Giamatti)

### Risk Profile
- **Franchise-First** — Built safety net before risks (e.g., Chris Evans post-Captain America)
- **Indie-First** — Built cred before mainstream (e.g., Adam Driver)
- **Balanced Portfolio** — Alternates deliberately (e.g., Florence Pugh)
- **Chaos Agent** — Unpredictable choices (e.g., Nicolas Cage)

### Comparable Mapping Questions
1. What actor 5-10 years ahead has a similar profile? What happened to them?
2. What actor with a similar breakthrough role plateaued? Why?
3. What actor successfully made the transition this actor is attempting?
4. What actor failed at the same transition? What warning signs did they show?

## Application
Use comparables to:
- Ground speculative claims in historical precedent (Tier B instead of C)
- Identify warning signs that predicted comparable failures
- Identify success patterns from comparable triumphs
- Flag when an actor is attempting something with no successful precedent
