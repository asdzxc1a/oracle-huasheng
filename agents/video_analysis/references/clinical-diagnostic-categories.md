# Clinical Diagnostic Categories — Video Analysis Agent

## Framework: Observable Behavioral Patterns

**Important:** This is not a clinical diagnosis. These are observational categories
used to describe behavioral patterns visible in video evidence. No agent claims
to diagnose any person with any condition.

## Categories

### Attachment Style (Observable Markers)
- **Secure:** Comfortable with intimacy, handles conflict directly, recovers quickly from stress
- **Anxious:** Seeks reassurance, over-explains, hypervigilant to interviewer reactions
- **Avoidant:** Deflects personal questions, intellectualizes emotion, maintains distance
- **Disorganized:** Inconsistent patterns, contradictory signals simultaneously

### Stress Response Pattern
- **Fight:** Becomes dominant, argumentative, competitive under pressure
- **Flight:** Deflects, changes subject, becomes vague
- **Freeze:** Goes blank, loses access to vocabulary, visible shutdown
- **Fawn:** Over-accommodates, agrees with everything, loses own position

### Narcissistic Spectrum (Adaptive vs. Maladaptive)
- **Adaptive:** High self-regard, uses charisma effectively, inspires loyalty
- **Maladaptive:** Requires constant validation, cannot handle criticism, exploits relationships
- **Vulnerable:** Fragile self-esteem masked by grandiosity, collapses under scrutiny

### Emotional Regulation
- **High:** Expresses emotion appropriately, recovers quickly, maintains boundaries
- **Moderate:** Some dysregulation under stress, generally functional
- **Low:** Frequent emotional flooding, poor recovery, boundary confusion

## Application to Casting
- Lead roles require moderate-to-high emotional regulation
- Villains often benefit from adaptive narcissistic traits
- Romantic leads need secure attachment markers (or convincing performance of them)
- Action heroes benefit from fight-pattern stress responses
