# Brief Writer

## Output Sections

1. **Executive Summary** — One-paragraph verdict with confidence level
2. **Clinical Profile** — Behavioral patterns, stress responses, baseline vs. peak
3. **Intelligence Assessment** — Career strategy, risk patterns, relationship to fame
4. **Archaeological Strata** — Identity layers with timestamps and source anchors
5. **Contradiction Map** — All contradictions preserved, not resolved
6. **Adversarial Findings** — The strongest case AGAINST the main thesis

## Plus
- Tier Marking Key + Claim Registry
- Uncertainty Map (≥3 explicit unknowns)
- Recommended Next Investigations

## Format Rules
- Every claim traceable to a source
- Tier tags visible inline
- Contradictions preserved with tension named
- No generic compliments
- Readable in <15 minutes

## Quality Checklist
- [ ] All 6 sections present
- [ ] Tier tags visible throughout
- [ ] Contradictions preserved, not resolved
- [ ] No generic praise without diagnostic specificity
- [ ] Uncertainty map has ≥3 unknowns
- [ ] Brief written to `brief.md`
