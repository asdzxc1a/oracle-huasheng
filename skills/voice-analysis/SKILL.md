# Voice Analysis

## Quick Start
For any audio/video source, analyze:
1. Baseline vocal register (relaxed conversation)
2. Stress-induced shifts (pitch, pace, volume changes)
3. Authenticity markers (hesitation patterns, filler words)
4. Emotional range (can they modulate voice for different affects?)

## Signal Categories

| Category | What to Listen For | Diagnostic Value |
|---|---|---|
| Baseline Register | Natural pitch, speaking pace, rhythm | Establishes vocal truth baseline |
| Stress Shifts | Pitch rise, speed increase, volume drop | Reveals authentic discomfort |
| Authenticity Markers | Appropriate hesitation, natural fillers | Indicates unscripted response |
| Emotional Range | Modulation for different emotional states | Predicts dramatic versatility |

## Access Level Impact
- **RAW sources**: Baseline is authentic; stress responses are genuine
- **MANAGED sources**: May show coached vocal patterns; look for cracks
- **SCRIPTED sources**: Only useful for vocal technique, not authenticity

## Quality Checklist
- [ ] Baseline established from relaxed conversation
- [ ] Stress shifts mapped to specific questions/moments
- [ ] Emotional range assessed across ≥2 emotional contexts
- [ ] All claims tagged with access level and tier
